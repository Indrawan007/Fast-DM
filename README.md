# Fast-DM

Fast-DM adalah aplikasi **Download Manager untuk Linux** dengan dukungan **browser extension** untuk mempermudah pengiriman link download ke aplikasi.

## Download
👉 https://github.com/Indrawan007/Fast-DM/releases/tag/v.1.1.0

## Release Files
- `fast-dm_1.0.0_amd64.deb` — aplikasi Linux
- `fast-dm-extension-v1.0.0.zip` — browser extension

## Instalasi

### Linux App
```bash
sudo apt install ./fast-dm_1.0.0_amd64.deb
